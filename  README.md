# Workout Health Analysis

The goal of this project is to develop a system that combines the technological aspects of both computer vision and machine learning to analyse an individuals form whilst preforming weighted workouts/exercise to identify any improper techniques or risks that a user can occur and provide live efficient feedback in real time with the use of a webcam.

## Features to be implemented

- Pose detection with the use of Mediapipe which detects up to 33 landmarks of the human body which can be further used to identify key body locations, analyze posture, and categorize movements.
- Feature extraction 
- Machine Learning Classification 
- Real-time feedback
- Rep Counter

## Project Structure

- src/ → core scripts
- data/ → datasets
- models/ → trained models
